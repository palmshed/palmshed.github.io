About

Palmshed is an open source software organization based in Kolkata, India. We build native and mobile applications, developer tools, SDKs and libraries, AI and machine learning systems, intelligent coding agents, and infrastructure. Our work also spans several focused organizations covering AI, design, libraries, research, systems, and developer tooling.
